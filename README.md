# Simpel Internet Connection Monitor
Simpel Internet Connection Monitor  adalah sebuah tools berbasis HTML yang akan memudahkan hidup anda dan bisa anda planjer atau nyalakan terus di browser.

# Cara pakai
- Simpan file ini sebagai baba-internet-monitor.html. atau apapun yg penting .html <br>
- Buka di browser desktop (Chrome/Edge/Firefox). Beri satu klik di halaman agar izin audio aktif.<br>
- Biarkan tab terbuka. Tool akan memantau koneksi dan berbicara otomatis.<br>

# Fitur ringkas
- Deteksi real-time: Event online/offline dari browser + ping berkala untuk verifikasi.<br>
- Suara: Web Speech API (speechSynthesis) dengan fallback nada beep jika speech tidak tersedia.<br>
- Anti-spam: Pesan suara tidak diulang jika status belum berubah.<br>
- Log & timestamp: Riwayat status dan waktu terakhir terdeteksi.<br>
- Ringkas & mandiri: Satu file, tanpa dependensi eksternal.<br>
